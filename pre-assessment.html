Software Engineering Infrastructure

Describe the purpose of an Interactive Development Environment (IDE):
To make it easier to write code.
Describe two advantages of an Interactive Development Environment:
The IDE can determine syntax errors in your code without having to run it. Keyboard Shortcuts and Extensions improve productivity and quality of life.
Define the purpose of a Source control:
To take snapshots and make changes to code repositories.
Describe two advantages of utilizing a source control system to develop software:
Previous commits can be restored to fix broken projects. Version control allows a team of developers to collaborate on a project.

Software Engineering Practice

What three programming languages are required to generate a web page:
Javascript, HTML, CSS
What is the purpose of writing pseudocode in software engineering:
To determine the logic and flow control needed to solve the problem before thinking about the syntax.
Write pseudocode for a for loop that calculates the sum of all integers from 0 to 9:
Create a Function; Inside that function initialize an empty variable to hold the sum, below create a for loop that iterates from zero to nine and adds the current counter to the sum variable on each iteration, then return the variable containing the sum.
Describe the purpose of a function in software engineering:
A reusable block of code that can be invoked.
Describe the purpose of a library in software engineering:
A collection of code such as functions or classes that serve a specific purpose so that developer does not need to re-invent the wheel.
